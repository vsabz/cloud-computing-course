GEMS-TUM

Graduate Employment Management System - TUM

About
The main purpose of the system is to provide the end user with analytics on the graduates’ employment rate and other parameters according to user query. The system solve the following tasks:
  • Graduates’ data generation: random generation, storing and provision (by request) of the graduates’ data using pre-formed lists of birth dates and names and other lists of data;
  • Query construction: creation of query to the graduates data storage, acquisition of the needed graduates’ data, formation of packages with graduates data to request employment of certain categories of graduates, sending the packages to the data processing application with the storage on the employment for all people;
  • Employment data generation : random generation, storing and provision (by request) of the people employment data using pre-formed lists of birth dates and names and constraints on salary levels;
  • Analytics calculation: acquisition of the employment and salary data as a result of request, calculation of meaningful parameters of employment for groups of graduates, and presentation of resulting data in a form of tables and graphs.
